Objetivo

Aplicar regressão linear múltipla para analisar o impacto conjunto dos investimentos em TV, rádio e jornal sobre o volume de vendas, ampliando a análise realizada anteriormente com regressão linear simples.

Dataset

Dataset público de publicidade e vendas, contendo informações sobre investimentos em TV, rádio e jornal e os respectivos volumes de vendas.

Dados disponíveis em:
https://www.kaggle.com/datasets/ashydv/advertising-dataset

Metodologia

Seleção das variáveis de investimento em TV, rádio e jornal como features

Separação dos dados em treino e teste

Treinamento de um modelo de Regressão Linear Múltipla

Avaliação do modelo com métricas de regressão

Análise dos coeficientes do modelo

Resultados

O modelo apresentou:

R² ≈ 0.89, indicando maior capacidade de explicação das vendas

Mean Squared Error (MSE) ≈ 3.17, com redução do erro em relação ao modelo simples

Os resultados mostram melhora clara no desempenho ao incluir múltiplas variáveis no modelo.

Conclusão

A regressão linear múltipla permitiu obter previsões mais precisas e uma compreensão mais completa do impacto dos diferentes canais de publicidade nas vendas, reforçando a importância do uso de múltiplas variáveis em problemas de regressão.
